NAME

Net::IRC::DCC - Object-oriented interface to a single DCC connection

SYNOPSIS

Hard hat area: This section under construction.

DESCRIPTION

This documentation is a subset of the main Net::IRC documentation. If you haven't already, please ``perldoc Net::IRC'' before continuing.

Net::IRC::DCC defines a few subclasses that handle DCC CHAT, GET, and SEND requests for inter-client communication. DCC objects are created by Connection-&gt;new_{chat,get,send}() in much the same way that IRC-&gt;newconn() creates a new connection object.
